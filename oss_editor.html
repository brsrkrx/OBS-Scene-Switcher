Joystick.tv Configuration

General information

Joystick.tv requires the WebSocket library. Good news: The server script can install it automatically!

✨ Automatic Installation (Recommended)

When you run oss_server.js with Joystick.tv enabled, if the WebSocket library is missing, the server will:

  1. Detect that it's not installed
  2. Ask: "Install WebSocket library now? (Y/n)"
  3. Install it automatically when you press Y
  4. Continue starting the server

Nothing to do in advance! Just run the server and let it handle it. 🎉

📘 Manual Installation (Optional)

If you prefer to install it manually first, navigate to the project folder and run:

npm install ws

Works on Mac, Windows, Linux - but remember it must be in the same folder as oss_server.js

Step 1: Create Bot Application

Go to joystick.tv/applications/new to create a new bot application

Bot Settings:
  • Name: OBS Scene Switcher
  • Description: Automatically switches OBS scenes based on tips
  • OAuth Redirect URL: http://localhost:3000/callback
⚠️ Required Permissions:
  • ✅ ReceiveStreamEvents ← Must enable!
  • ✅ ViewUserPresence (optional)
  • ⭕ All others (not needed)

Step 2: Get Credentials

Go to joystick.tv/applications, scroll down to find the bot you created in step 1, click Options → Settings to view your credentials.

Copy from bot settings page
Copy from bot settings page (keep this secret!)

Step 3: Install Bot on Your Channel

At joystick.tv/applications, find your bot and click "Install". Authorize the permissions when prompted.

💡 The OAuth Redirect URL is required by Joystick.tv but won't be used by our bot. We connect directly using Client ID + Secret.
⚠️ Please enable at least one platform and configure its credentials
Instructions
  1. Select which platform(s) to use above
  2. Configure credentials for enabled platform(s)
  3. Click "Download config.json" - saves your configuration file
  4. Place config.json in the same folder as oss_server.js
  5. Start the server (double-click START_SERVER_WINDOWS.bat on Windows or START_SERVER_MAC.command on Mac) or run: node oss_server.js
  6. Make sure you have set up the configuration for scene switching, and applied it. Send some test tips to trigger switch so you know the changes are like you want
  7. Start streaming!
💡 The server automatically reads config.json on startup. For Joystick.tv users, the WebSocket library installation is fully automated - the server script handles it for you. Remember not to share the config.json file with anyone!

OBS Scene Switcher v1.0

Beautiful and user-friendly scene switcher with customizable overlay for streaming platforms. Features live preview with animated demonstrations, preset themes, custom theme management, scene queue system, and comprehensive customization options.

Key Features

  • Live Preview when editing themes - See changes instantly with animated previews
  • Resizable Panels - Drag divider to adjust preview size
  • 4 Theme Presets - Cyberpunk, Minimal, Retro Wave, Luxury Gold
  • Custom Themes - Save, load, export, and import unlimited themes
  • Visual Gradient Builder - No CSS knowledge required
  • 6 Animation Styles - Slide, Fade, Zoom, Bounce, Stamp, Flip
  • 7 Position Options - Place alerts anywhere on screen
  • Dual Opacity Controls - Separate box and background transparency
  • Complete Typography Control - Individual font sizes and families
  • Per-Scene Return Timers - Different delays for each tip amount
  • Scene Return Progress Bar - Visual countdown at screen bottom
  • Smart Text Color - Automatic contrast for light backgrounds
  • Server Config Generator - Download ready-to-use config file
  • Per-scene overlay toggle - show/hide overlay for specific scenes
  • Global overlay toggle - for non-scene switching tips
  • Custom test tip editor - test with any username, amount, and message
  • Scene Change Queue - Scene switches queue up automatically, no overlaps

Quick Start

  1. Add oss_overlay.html to OBS as Browser Source. You can drag and drop it, just make sure it is at the top or at least over your video sources
  2. Open properties of oss_overlay.html and enable "Advanced access to OBS" in page permissions
  3. Go to "Server Setup" tab and configure your streaming platforms
  4. Download configuration file and place it in the same folder where this file is
  5. Start the server. Easiest way is to double click START_SERVER_WINDOWS.bat on Windows, and START_SERVER_MAC.command on Mac
  6. Customize theme and configuration in this editor
  7. Click "Apply Changes to OBS" to push changes to OBS
  8. Click "Send Test Tip" to preview scene switch and animations in OBS

Supported Platforms

  • Chaturbate - Connects via Events API ✓
  • Joystick.tv - Connects through WebSocket API ✓

Tips

  • You don't need to have the editor (this page) open all the time, only when making changes
  • The overlay toggle controls which scenes show overlays
  • Test Editor - Click ⚙️ Edit Test Tip to customize test data
  • Resizer - Drag divider to adjust size, the preview auto-scales
  • Progress Bar - Empties as time runs out, customizable thickness
  • Scene Names - Must match scene names in OBS exactly (capitals matter!). Can be imported from OBS when all is connected
